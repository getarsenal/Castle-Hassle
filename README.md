# Castle Hassle

A 2.5D medieval **siege** game for iOS (and the web). Command an army of
thousands storming a castle — flood the gate and the breach, put archers to
work, and break the garrison.

Built as a web game (TypeScript + three.js + Vite), shipped to the App Store via
Capacitor + Codemagic. Soft, chunky, flat-shaded art; ~2,200 soldiers on screen.

```bash
npm install
npm run dev      # play at http://localhost:5173
```
