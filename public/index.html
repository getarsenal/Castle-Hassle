📱 iOS (Shortcuts) — auto-changes every morning
  1. Open Shortcuts → Automation → + → Time of Day, pick a time (e.g. 6:00 AM), Daily, Run Immediately.
  2. Add action Get Contents of URL → paste the wallpaper URL above.
  3. Add action Set Wallpaper Photo → use the downloaded image; turn off "Show Preview".
  4. Done. Every morning your Home & Lock screen becomes the new horror.
🤖 Android

Use an app like Tasker or a live-wallpaper client that fetches the URL daily, or the (planned) native Castle Hassle app which sets the wallpaper via WallpaperManager on a daily background job.
